# 인도 소비재: 모디노믹스 패러다임 전환에 투자하라

## 투자 개요

**모디노믹스 3기, 소비를 깨우다** : 모디 정부는 2014년 출범 이후 각 임기마다 뚜렷이 구분되는 경제 정책 기조를 펼쳐왔다. 1기(2014~2019년)에는 제도 개혁과 경제 체질 개선을 통해 성장 기반을 다지는 데 주력했고, 2기(2019~2024년)에는 대규모 인프라 투자와 공급 기반 확충에 정책 우선순위를 두었다. 3기(2024~2029년) 정부는 소득세 인하, 부가가치세 인하(GST 단순화), 금리 인하 등을 통해 소비 진작에 역점을 두고 있다. 특히 2025년 4분기와 2026년에 예정된 주요 선거를 앞두고 지지 기반을 강화하기 위해 이러한 정책 기조를 더욱 강화할 것으로 전망된다.

- 정부 소비 진작 정책: CY4Q25부터 기업 이익에 반영 전망
- 소비 관련 매크로 펀더멘털 개선, 인도 내수 회복의 순풍
- FY27년 임금 인상 수혜: 전체 가구의 7~8%→ 내수 소비 자극
- 밸류에이션: 부담스럽지 않은 수준

## 모디노믹스 1기 (2014~2019): 제도 개혁과 토대 구축기

**정책 기조** : 경제 기반 정비와 시스템 개혁

- **핵심 정책**
    - GST(부가가치세) 도입(2017) → 통합 내수 시장 형성
    - 고액권 폐지(2016) → 암거래·탈세 억제
    - 금융포용 정책(Jan Dhan Yojana) → 은행 계좌 개설 확대, 디지털 지급·보조금 송금 활성화
- **정책 특징** : 소비 촉진보다는 제도적 정비와 시스템 개혁에 우선순위
- **종합 평가** : '제도적 토대 구축기' → 향후 인프라·제조업 투자 및 소비 진작의 발판
- **주식시장 성과**
    - Nifty50 지수: 연평균 11.9%
    - 금융 (Nifty Bank: 18.3%, Nifty Financial Services: 18.8%)→ 가장 두드러진 성과
    - 생활소비재 (Nifty FMCG: 11.5%) → Nifty50 소폭 하회


**그림 1. 모디노믹스 1기 경제정책 수익률 효과** : Nifty50(11.9%), 은행(18.3%), 금융서비스(18.8%), IT(12.6%), 생활소비재(11.5%), 자동차(7.3%)

**그림 2. 모디노믹스 1기 경제정책의 업종별 누적 수익률 비교** : Nifty50(11.9%), 은행(181%), 금융서비스(175.4%), IT(181%), 생활소비재(142.3%), 자동차(50%)

**그림 3. 모디노믹스 1기 업종별 주가 성과 추이(2014~2019)** : 금융, IT 업종 강세, 생활소비재, 자동차 업종은 Nifty50와 유사한 수준

**표 2. 모디노믹스 1기(2014~2019): 제도 개혁과 업종별 주식시장 성과** : 정책 기조, 핵심 정책, 정책 특징, 종합 평가, 주식시장 성과 요약


## 모디노믹스 2기 (2019~2024): 공급 기반 확충과 인프라 드라이브

**정책 기조** : 제조업과 인프라 확충에 집중

- **핵심 정책**
    - 대규모 인프라 투자 (도로·철도·항만·공항 등) → 고용 창출 및 생산성 향상
    - PLI(생산연계 인센티브) 제도 → 휴대폰, 전기차 배터리, 반도체 등 전략 산업 육성
    - 디지털 인프라 확충→ UPI 전국 확산, 핀테크 성장, 보조금 지급 전자화
- **정책 특징** : 내수 소비 자극보다는 생산 기반 강화·공급 측 구조 확대에 방점
- **종합 평가** : '인프라·제조업 중심 성장기' → 글로벌 공급망 내 위상 강화, 단기 체감 경기 회복은 제한적
- **주식시장 성과**
    - Nifty50 지수: 연평균 14.0%
    - 부동산 (Nifty Realty: 30.4%) → 가장 높은 성과
    - 금속 (Nifty Metal: 24.3%)
    - 정유·화학 (Nifty Oil & Gas: 23.1%)
    - 자동차 (Nifty Auto: 21.9%)
    - 인프라 (Nifty Infrastructure: 22.5%)
    - 생활소비재 (Nifty FMCG: 12.3%) → Nifty50 하회


**그림 4. 모디노믹스 2기: 공급 드라이브가 이끈 업종별 주가 성과** : 인프라(22.5%), 부동산(30.4%), 금속(24.3%), 정유·화학(23.1%), 자동차(21.9%), 생활소비재(12.3%), Nifty50(14%)

**그림 5. 모디노믹스 2기 경제정책의 업종별 누적 수익률 비교** : 부동산(376.8%), 금속(297%), 정유·화학(276%), 인프라(225%), 자동차(178.8%), 생활소비재(12.3%), Nifty50(100%)

**그림 6. 모디노믹스 2기 업종별 주가 성과 추이(2019~2024)** : 부동산, 금속, 정유·화학, 인프라, 자동차 업종 강세, 생활소비재 업종은 Nifty50와 유사한 수준

**표 3. 모디노믹스 2기(2019~2024): 인프라 드라이브와 업종별 시장 성과** : 정책 기조, 핵심 정책, 정책 특징, 종합 평가, 주식시장 성과 요약


## 모디노믹스 3기 전반 (2024~2026): 소비 진작과 내수 부양 모드

**정책 기조** : 소비 내수 회복에 집중

- **핵심 정책**
    - 소득세 인하(2025.2) → 중산층 가처분 소득 확대
    - 금리 인하(2025.3) → 대출 비용 절감, 소비·투자 촉진
    - GST 체계 단순화·세율 인하(2025.8) → 소비자·기업 세 부담 완화
    - 보조금·대출 지원·농촌 프로그램 강화 → 농촌 소득 개선 병행 (예상)
- **정책 특징** : 단기 소비 여력 확대, 내수 시장 활성화에 초점
- **종합 평가** : '소비 진작 모드' → 국민 생활과 직접 연결된 경기 부양
- **주식시장 성과(25년 8월 25일 기준)**
    - Nifty50 지수: 연평균 7.0%
    - 금융 서비스 (Nifty Financial Services: 13.9%) → 가장 두드러진 성과
    - 자동차 (Nifty Auto: 9.4%)
    - 생활소비재 (Nifty FMCG: 2.7%)


**그림 7. 모디노믹스 3기 전반: 소비 진작 정책과 업종별 주가 성과** : 금융 서비스(18.8%), 자동차(9.4%), 생활소비재(2.7%), 은행(7.6%), IT 서비스(7%), 철강(6.4%), 정유·화학(3.4%), 부동산(3.6%), 인프라(1.1%)

**그림 8. 모디노믹스 3기 전반 누적 수익률 비교** : 금융 서비스(13.9%), 자동차(9.3%), 은행(10.2%), IT 서비스(8.6%), 철강(4.6%), 생활소비재(1.1%), 정유·화학(3.6%), 부동산(0.9%), 인프라(-5.6%)

**그림 9. 모디노믹스 3기 업종별 주가 성과 추이(2024~)** : 금융 서비스 업종 강세

**표 4. 모디노믹스 3기 전반(2024~2026): 소비 진작과 내수 부양 모드** : 정책 기조, 핵심 정책, 정책 특징, 종합 평가, 주식시장 성과 요약


## 모디노믹스 3기 후반 (2027~2029): 재정 제약과 성장 동력의 재조정기

**정책 기조** : 공급 기반 강화와 인프라 투자 재부각

- **핵심 정책**
    - 재정 관리 강화 → 소득세 인하·GST 조정 등 감세 효과로 약화된 중앙·주정부 세수 보완 필요
    - 투자 중심 전환 → 인프라·제조업 프로젝트로 재정지출 효율성 제고
    - PLI 정책·인프라 프로젝트 심화 → 글로벌 공급망 재편·중국 둔화 활용
- **정책 특징** : 소비 의존 한계를 인식하고 구조적 성장 동력(제조업·수출·SOC 확충) 재부각
- **종합 평가** : '재정 제약 속 균형 모색기' → 단기적 소비 자극에서 벗어나 중장기 성장 경로 안정화
- **주식시장 성과(추정)** : 모디 2기와 유사하게 인프라·부동산 업종 강세 지속 전망


**표 5. 모디노믹스 3기 후반(2027~2029): 재정 제약과 성장 동력의 재조정기** : 정책 기조, 핵심 정책, 정책 특징, 종합 평가 요약

**표 6. 모디노믹스 1~3기: 정책 기조와 업종별 주식시장 성과 종합 비교** : 모디노믹스 1,2,3기 정책 기조, 핵심 정책, 정책 특징, 종합 평가, 주식시장 성과 요약
